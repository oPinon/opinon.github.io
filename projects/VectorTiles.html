Rendering a map as vector tiles for the web

Note : Made in collaboration with Radio-Canada and École Polytechnique de Montréal

.

We created several components to display the map :

  • A C++ program that converts the raw map to a tree of vector tiles (as .json files)
  • A simple server that hosts the tiles
  • A plugin for Leaflet that displays the tiles.
The polygons can then be stylized using any CSS (they are SVGs) and D3.js if there is some data.
A User Guide and some samples are available here.
